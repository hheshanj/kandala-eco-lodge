# Galpothuyaya Eco Lodge — Website Project

A premium, responsive, and immersive multi-page website for **Galpothuyaya Eco Lodge**, an intimate wilderness retreat situated on the banks of Kandalama Lake in Sri Lanka's Cultural Triangle.

## 🌐 Live Preview
The website is professionally deployed and can be accessed at:
**[https://eco-lodge.netlify.app](https://eco-lodge.netlify.app)**

---

## 🍃 Project Overview
This website was designed to capture the serene and "petite" essence of Galpothuyaya. It focuses on high-quality visuals, smooth interactions, and a seamless user journey that guides potential guests from discovery to booking.

### Key Features
- **Modern & Editorial Design**: A unique, asymmetrical layout that feels premium and organic.
- **Fully Responsive**: Optimized for all devices, from high-resolution desktops to mobile smartphones.
- **Immersive Visuals**: High-resolution imagery and subtle scroll-reveal animations (using Intersection Observer API).
- **Comprehensive Content**:
  - **Home**: Narrative introduction to the lodge and its unique setting.
  - **Rooms**: detailed showcase of accommodation types including the signature Tree House.
  - **Retreats**: Specialized sections for wellness, creative, and corporate residencies.
  - **Explore**: Integrated guide to local attractions (Sigiriya, Dambulla Lake).
  - **Dining**: A window into the authentic Sri Lankan culinary experience.
  - **Contact**: User-friendly inquiry and reservation system.

---

## 🛠️ Technical Implementation
The project is built using modern web standards for maximum performance and SEO efficiency.

- **Frontend**: Semantic HTML5, Vanilla CSS3 (Custom Properties / Flexbox / Grid), and Vanilla JavaScript.
- **Typography**: Curated font pairings (Cormorant Garamond for elegance, Jost for modern readability) via Google Fonts.
- **Animations**: Custom CSS transitions and JavaScript-driven scroll reveal effects.
- **Performance**: Optimized for fast loading with deferred scripts and lazy-loading images.
